The canvas window shows PHYSVIS objects in 3D.
The origin of the canvas has coordinates (0,0,0) in a right-handed coordinate system.
As you work in MATLAB, you issue commands that create PHYSVIS visual objects and display them in the canvas. For example, create a box named mybox by typing this statement at the command line:
mybox = physvis.Box('Position',[0 0 0],'Size',[10 5 3],'Color',Colors.red)

In the canvas window, click and drag with the right mouse button to rotate the camera. Scroll the mouse wheel to zoom the camera in and out. Click and drag with the left mouse button to pan the camera.
MATLAB adds object mybox to the workspace and displays the result in the Command Window. You can see the available properties of mybox. Change the color property of mybox by typing this statement at the command line:
mybox.Color = Colors.blue
mybox =
Box with properties: Length: 10 Height: 5 Width: 3 Size: [10 5 3] Retain: 10 HasTrail: 0 TrailType: 'points' TrailRadius: 0.1250 Axis: [1 0 0] Up: [0 1 0] TrailObject: [] IdentityMatrix: [4×4 double] Color: [0 0 1] Canvas: [1×1 physvis.Canvas] Alpha: 1 Red: 0 Green: 0 Blue: 1 Visible: 'on' UserData: [] X: 0 Y: 0 Z: 0 Position: [0 0 0]
If you end a statement with a semicolon, MATLAB performs the computation, but suppresses the display of output in the Command Window.
mybox.Color = Colors.green;
You can recall previous commands by pressing the up- and down-arrow keys. Press the arrow keys either at an empty command line or after you type the first few characters of a command. For example, to recall the command mybox = physvis.Box(...), type mybox =, and then press the up-arrow key.
