Elastic Collisions of a Ball in a Box

Open Live Script

Background

Perfectly elastic collisions do not result in any loss in energy. In this example, we will explore what happens when a ball is thrown inside of a box, first with perfectly elastic collisions and then under the influence of gravity.

Setup

First we will construct the experimental setup, which consists of 5 walls of a box, and a ball. This defines the size of the box:
distToSide = 4.0; % meters
wallThickness = 0.3; % meters
shortSideLength = 2*distToSide - wallThickness;
longSideLength = 2*distToSide + wallThickness;
This creates the walls of the box and stores them in a structure called walls:
walls.Right = physvis.Box('Position', [distToSide, 0, 0], ...
'Size', [wallThickness, shortSideLength, longSideLength], ...
'Color', Colors.red);
walls.Left = physvis.Box('Position', [-distToSide, 0, 0], ...
'Size', [wallThickness, shortSideLength, longSideLength], ...
'Color', Colors.red);
walls.Bottom = physvis.Box('Position', [0, -distToSide, 0], ...
'Size', [longSideLength, wallThickness, longSideLength], ...
'Color', Colors.blue);
walls.Top = physvis.Box('Position', [0, distToSide, 0], ...
'Size', [longSideLength, wallThickness, longSideLength], ...
'Color', Colors.blue);
walls.Back = physvis.Box('Position', [0, 0, -distToSide], ...
'Size', [shortSideLength, shortSideLength, wallThickness], ...
'Color', [0.7, 0.7, 0.7]);
Now we define a green ball ball with a radius of 0.4 meters, that will have a 3D curve tracing out its path, showing the previous 1000 positions:
ball = physvis.Sphere('Color', Colors.green, 'Radius', 0.4, ...
'HasTrail', true, 'Retain', 1000, 'TrailType', 'curve');
Then we define the ball's mass mass and momentum p:
ball.UserData.mass = 1.0; % kg
ball.UserData.p = vector(-0.15, -0.23, +0.27); % m/s
This line subtracts out the radius of the ball and half the wall thickness from the distance to the side of the box, since we will consider the ball to be in contact with the wall once this distance is reached.
distToSide = distToSide - wallThickness*0.5 - [ball.Radius];

Simulation Loop

Once we have setup the experimental environment, we can run a loop to explore the physics.
% Define our time step:
deltaTime = 0.3; % seconds
We will simulate 5 minutes of bouncing (1000 itterations X 0.3 seconds = 300 seconds X 1 minute / 60 seconds = 5 minutes)
for iteration = 1:1000
First update the position of the ball in this time step based on its momentum and mass.
ball.Position = ball.Position + (ball.UserData.p/ball.UserData.mass)'*deltaTime;
If the ball has made contact with the wall in the x,y, or z directions, reverse the direction of the ball's momentum in that direction, simulating a perfectly elastic collision.
if ~(-distToSide < ball.X && ball.X < distToSide)
ball.UserData.p.x = -ball.UserData.p.x;
end
if ~(-distToSide < ball.Y && ball.Y < distToSide)
ball.UserData.p.y = -ball.UserData.p.y;
end
if ~(-distToSide < ball.Z && ball.Z < distToSide)
ball.UserData.p.z = -ball.UserData.p.z;
end
Define the frequency of the simulation in Hz. A value of would be real world time, and would take about 5 minutes to run. This simulation is running at 200 Hz, which means it should take about 5 seconds to run in the real world.
rate(200);
end

Adding Gravity

If we want to introduce some energy loss into the system we can do it by applying gravity. This time the time step will be reduced by an order of magnitude because the ball will be moving much faster than it was before.
deltaTime = 0.03;
We will also move the ball back to the center of the box, reset its trail, and change its momentum.
ball.Position = [0,0,0];
ball.HasTrail = false;
ball.HasTrail = true;
ball.UserData.p = vector(-1.5, -2.3, +2.7); % m/s
The beginning of the for loop stays the same as it was previously.
for iteration = 1:200
ball.Position = ball.Position + (ball.UserData.p/ball.UserData.mass)'*deltaTime;
if ~(-distToSide < ball.X && ball.X < distToSide)
ball.UserData.p.x = -ball.UserData.p.x;
end
if ~(-distToSide < ball.Y && ball.Y < distToSide)
ball.UserData.p.y = -ball.UserData.p.y;
end
if ~(-distToSide < ball.Z && ball.Z < distToSide)
ball.UserData.p.z = -ball.UserData.p.z;
end
After we check the ball's position, we will also add the effect of gravity
ballHeight = distToSide + ball.Y;
ball.UserData.p.y = ball.UserData.p.y - (ball.UserData.mass*9.81*deltaTime)*(ballHeight >= 0);
rate(200);
end
