Tumbling Box Example

Open live script
This example shows how to move a box along a 3D curve as it spins.
% Create a box that will rotate.
box = physvis.Box('Size', [1, 0.5, 0.5]);
box.Color = Colors.blue;
box.Position = [2,0,0];
box.HasTrail = true;
box.Retain = 500;
box.TrailRadius = 0.125;
% Rotate the box through a 3D path.
t = 0;
dt = pi/100;
while t < 2*pi %(isvalid(box.Canvas))
Rotate(box, dt, [sin(t),cos(t),0], [0,0,0]);
rate(60);
t = t+dt;
end
