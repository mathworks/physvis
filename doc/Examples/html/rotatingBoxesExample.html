Rotating Boxes Example

Open live script
This example shows how to animate the rotation of multiple boxes simultaneously.
N = 4;
boxes(N^3) = physvis.Box();
L = 6;
for x = 1:N
for y = 1:N
for z = 1:N
boxes(((x-1)*N+y-1)*N+z) = physvis.Box('Color', [x,y,z]./N, ...
'Position', [L*((x-1)/(N-1)-0.5), L*((y-1)/(N-1)-0.5), L*((z-1)/(N-1)-0.5)], ...
'Size', [0.6*L/N, 0.4*L/N, 0.6*L/N]);
end
end
end
t = 0;
dt = 0.01;
while t < 1 %(isvalid(boxes(1).Canvas))
t = t + dt;
Rotate(boxes, dt, [0,1,0])
rate(200);
end
